
# mtproxy

这是一个可以在serv00便用的 MTProxy 代理的绿色脚本

## 安装方式

执行如下代码进行安装

```bash
curl -s -O https://raw.githubusercontent.com/boosoyz/mtproto/main/mtg.sh

chmod +x mtg.sh

./mtg.sh
```

## 卸载安装

因为是绿色版卸载极其简单，直接删除所在目录即可。

```bash
rm -rf /home/${USER}/mtg
```

# 可自行选择保活

设置定时任务每13分钟执行一次
